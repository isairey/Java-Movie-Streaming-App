# 🌌 Acerca del proyecto

**Java Movie Streaming App** es una plataforma multimedia enfocada en la exploración y búsqueda avanzada de películas utilizando Java Servlets y bases de datos relacionales.

La aplicación fue desarrollada para practicar:

- ☕ Desarrollo backend con Java
- 🌐 Java Servlets
- 🗄️ Bases de datos MySQL
- ☁️ Deployment en AWS
- 🚀 Apache Tomcat
- 🎬 Plataformas multimedia

El sistema permite consultar películas mediante múltiples filtros y visualizar información multimedia dinámica.

# ⚡ Instalación

## 📋 Requisitos

- Java JDK
- Apache Tomcat
- MySQL
- AWS Account (opcional)

---

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/Java-Movie-Streaming-App.git
```

---

## 2️⃣ Configurar base de datos

- Crear base de datos MySQL
- Importar datos multimedia
- Configurar credenciales JDBC

---

## 3️⃣ Desplegar en Tomcat

Copiar proyecto al servidor Tomcat.

---

## 4️⃣ Ejecutar servidor

Iniciar Apache Tomcat:

```bash
startup.sh
```

o en Windows:

```bash
startup.bat
```
